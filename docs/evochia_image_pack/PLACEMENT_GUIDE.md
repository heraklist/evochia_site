## Practical notes
- Use landscape images for desktop sections, hero bands, and service cards.
- Use portrait images where the layout supports tall crops.
- Keep `alt` text from `manifest.csv` unless you want to tailor it to the exact on-page copy.
- Avoid using the same image in more than two sections.
