### 2.1 Βασικά ζητήματα

---

#### Hero title EL — απώλεια `<em>` italics
```
ΤΡΕΧΟΝ EN:  "Crafting <em>Extraordinary</em> Culinary Moments"  → Miama script στο "Extraordinary"
ΤΡΕΧΟΝ EL:  "Δημιουργώντας <em>εξαιρετικές</em> γαστρονομικές στιγμές"
             → Πέφτει πίσω στο Alexander font (ΣΩΣΤΑ — Miama δεν έχει ελληνικά)
             → Αλλά η λέξη χάνει τη visual emphasis που έχει στα αγγλικά
```
**Πρόταση:** Για να διατηρηθεί η οπτική αναγνωριμότητα στα ελληνικά `<em>`, εξέτασε:
- Χρήση `color: var(--gold)` — που ήδη εφαρμόζεται ✅
- Αύξηση `font-size` κατά 5-10% για τις EL em λέξεις
- Ή: χρήση του Bainsley **italic** αντί του Alexander για τις ελληνικές `<em>`, που θα διατηρεί διαφορετικό "χαρακτήρα" από το υπόλοιπο heading

## 3. ΕΜΦΑΝΙΣΗ

### 3.1 Philosophy ornament — χαμένη ευκαιρία Miama

```css
/* ΤΡΕΧΟΝ */
.philosophy-ornament {
  font-family: var(--font-heading);  /* Alexander */
  font-size: 4rem;
  color: var(--gold-dim);
}

/* ΠΡΟΤΕΙΝΩ — χρήση Miama για το decorative " */
.philosophy-ornament {
  font-family: var(--font-accent);  /* Miama script */
  font-size: 7rem;   /* μεγαλύτερο για impact */
  color: var(--gold-dim);
  line-height: .8;
  margin-bottom: .5rem;
  letter-spacing: -.02em;
}
```
**Γιατί:** Το `"` quotation mark σε Miama script font θα ήταν εντυπωσιακό decorative element — πολύ πιο ζωντανό από το Alexander serif. Αυτό είναι ακριβώς η χρήση για την οποία προορίζεται ένα script accent font.

### 3.2 Philosophy attribution — προσθήκη Miama

```html
<!-- ΤΡΕΧΟΝ -->
<cite class="philosophy-attribution">— Heraklis, Founder</cite>

<!-- ΠΡΟΤΕΙΝΩ — χωρίστε σε 2 μέρη για visual hierarchy -->
<cite class="philosophy-attribution">
  <span class="philosophy-attr-name">— Heraklis</span>
  <span class="philosophy-attr-title">Founder, Evochia</span>
</cite>
```
```css
.philosophy-attr-name {
  font-family: var(--font-accent); /* Miama */
  font-size: 1.3rem;
  color: var(--gold-light);
  display: block;
}
.philosophy-attr-title {
  font-size: .65rem;
  letter-spacing: .3em;
  text-transform: uppercase;
  color: var(--cream-dim);
  display: block;
  margin-top: .3rem;
}
```
**Γιατί:** Αυτή η αλλαγή κάνει τον Ηρακλή πιο παρόντα στη σελίδα — humanizes the brand χωρίς να χρειάζεσαι portfolio ή testimonials.

### 3.3 Gold line dividers — gradient upgrade

```css
/* ΤΡΕΧΟΝ */
.gold-line { width: 45px; height: 1px; background: var(--gold); margin-bottom: 2rem; }

/* ΠΡΟΤΕΙΝΩ — gradient fade για πιο εκλεπτυσμένη εμφάνιση */
```

**Γιατί:** Το solid 1px gold line είναι καλό αλλά flat. Το gradient fade δίνει κίνηση και lightness — αισθητικά πιο εκλεπτυσμένο χωρίς να αλλάζει η λογική.

### 3.4 Stats section — ανεπαρκής visual weight

```css
/* ΤΡΕΧΟΝ */
.stat-number { font-size: 3rem; color: var(--gold); }
```
**Πρόβλημα:** Το ✦ symbol στα 3rem είναι πολύ μικρό για να λειτουργήσει ως "number". Το 100% λειτουργεί καλύτερα.

**Πρόταση A** — χρήση αριθμών αν υπάρχουν real data:
```
Αντί για ✦ Always Bespoke → ∞ Always Bespoke
Αντί για ✦ Made to Order  → ✦ No Menu Repeated
```

**Πρόταση B** — visual upgrade χωρίς αριθμούς:
```css
/* Αύξησε το ✦ symbol */
.stat-number-symbol { font-size: 2.5rem; opacity: .7; }
/* Και δώσε περισσότερο βάρος στο label */
.stat-label { font-size: .75rem; color: var(--cream); letter-spacing: .15em; }
```

### 3.6 Venue cards — CTA always visible on mobile

```css
/* ΤΡΕΧΟΝ — CTA εμφανίζεται μόνο σε hover */
.venue-card-cta { opacity: 0; transform: translateY(8px); transition: all .3s; }
.venue-card:hover .venue-card-cta { opacity: 1; transform: translateY(0); }

/* ΠΡΟΤΕΙΝΩ — always visible σε mobile */
@media (hover: none) {
  .venue-card-cta { opacity: 1; transform: translateY(0); }
}
```
**Γιατί:** On touch devices, hover state δεν υπάρχει. Σε mobile οι χρήστες δεν βλέπουν ποτέ τα CTAs. Με `@media (hover: none)` — που στοχεύει ακριβώς touch screens — το CTA γίνεται always-visible χωρίς να επηρεαστεί το desktop.

## 4. ΠΑΡΟΥΣΙΑΣΗ

### 4.1 Homepage — σειρά sections

**Τρέχουσα σειρά:**
1. Hero
2. About
3. Services (tabs)
4. Cuisines
5. Philosophy quote ← **εδώ είναι λίγο νωρίς**
6. Stats
7. How It Works (practical trust)
8. Venues
9. CTA

**Πρόβλημα:** Η Philosophy quote (συναισθηματικό κλιμάκιο) εμφανίζεται στο μέσο της σελίδας, ανάμεσα σε Cuisines και Stats. Χάνει την force της γιατί δεν είναι σε climax position.

**Εναλλακτική σειρά:**
1. Hero
2. About
3. Services (tabs)
4. Cuisines
5. **How It Works** (practical — builds trust early)
6. **Stats** (validates brand promises)
7. **Philosophy quote** ← ως emotional apex πριν το final CTA
8. Venues
9. CTA

**Γιατί:** Η ακολουθία γίνεται: "Τι κάνουμε → Πώς → Γιατί υπάρχουμε → Πού πάμε → Ελάτε". Αυτός ο narrative arc είναι πιο πειστικός.

---

### 4.2 Trust section — υπερφόρτωση

Το "How It Works" section περιέχει αυτή τη στιγμή:
- 3-step process (01/02/03)
- 3 trust points (24hrs reply / Built for venue / Scaled to guest)
- 3 mini FAQ items

Αυτά είναι **9 elements** σε ένα section, χωρισμένα σε 2 panels. Είναι πολύ.

**Πρόταση:** Διάσπασε το trust section:
- **Panel A:** 3-step process (κρατά)
- **Panel B:** 3 trust points (κρατά)
- **Αφαίρεσε** τα mini FAQ items από εδώ → μεταφόρτωσέ τα σε slim FAQ strip κάτω από το section με link "See all FAQs →"

Αποτέλεσμα: Πιο αέρινο section, και τα FAQ items αποκτούν link που θα οδηγεί στη νέα FAQ page.

---

### 4.3 Cuisines section — μόνο 3 cuisines στο homepage

## 6. ΧΡΗΣΗ ΓΡΑΜΜΑΤΟΣΕΙΡΩΝ

### 6.1 Σύνοψη τρέχουσας χρήσης

| Font | Χρήση | Αξιολόγηση |
|------|-------|------------|
| **Alexander** (serif) | h1, h2, h3, nav-logo, philosophy-ornament | ✅ Εξαιρετική |
| **Bainsley** (body) | body, hero-tagline (italic), nav-links, buttons, labels | ✅ Εξαιρετική — 4 variants χρησιμοποιούνται |
| **Miama** (script) | `<em>` μέσα σε hero-title, section-title, service h3, cta-title | ✅ Έξυπνη χρήση αλλά μπορεί να επεκταθεί |

---

### 6.2 Alexander — χρήση μόνο σε weight 400

Το Alexander φορτώνεται μόνο σε 400 weight. Αυτό είναι σωστό (μία μόνο περικοπή = performance). Αλλά:

**Παρατήρηση:** Στα headings, η εναλλαγή μεταξύ Alexander (upright) και Miama (italic/script) για τις `<em>` λέξεις λειτουργεί εξαιρετικά — δίνει ρυθμό χωρίς να χρειαστεί bold.

**Πρόταση:** Χρησιμοποίησε Alexander και για το footer brand name:
```css
/* ΤΡΕΧΟΝ — ήδη υπάρχει */
.footer-brand { font-family: var(--font-heading); }
/* ✅ Ήδη σωστό */
```

---

### 6.3 Miama — υπο-χρησιμοποιημένη σε 3 ευκαιρίες

**Τρέχον:** Χρησιμοποιείται αποκλειστικά για `<em>` words σε headings (hero-title, section-title, service h3, cta-title).

**Ευκαιρίες επέκτασης:**

#### A. Philosophy ornament
```css
/* Από Alexander → Miama για μεγαλύτερο visual impact */
.philosophy-ornament { font-family: var(--font-accent); font-size: 7rem; }
```

#### B. Philosophy attribution
```css
/* "— Heraklis" σε Miama */
.philosophy-attribution .name-part { font-family: var(--font-accent); font-size: 1.4rem; }
```

#### C. Footer tagline
```css
/* Ελαφρά Miama touch στο footer tagline */
.footer-tagline { font-family: var(--font-accent); font-size: 1.1rem; letter-spacing: .02em; }
```

#### D. Conciergerie label (αν προστεθεί)
```css
.conciergerie-label { font-family: var(--font-accent); font-size: 0.9rem; }
```

---

### 6.4 Bainsley italic — χρήση στο hero-tagline

```css
.hero-tagline { font-style: italic; }  /* Bainsley italic */
```
**Αξιολόγηση:** ✅ Εξαιρετική χρήση — το italic δίνει softer, more personal tone στη tagline.

**Πρόταση:** Χρήση Bainsley italic και στις `<cite>` elements (philosophy attribution, testimonials αν προστεθούν).

### 6.5 Greek language — font fallback για em

```css
/* ΤΡΕΧΟΝ — σωστό fallback: Miama → Alexander για ελληνικά */
html[lang="el"] .hero-title em,
html[lang="el"] .section-title em {
  font-family: var(--font-heading); /* Alexander */
}
```

**Πρόταση βελτίωσης:** Αντί για Alexander (που δεν υποστηρίζει ελληνικούς χαρακτήρες και θα πέφτει σε Georgia/serif), χρησιμοποίησε:
```css
html[lang="el"] .hero-title em,
html[lang="el"] .section-title em {
  font-family: var(--font-body); /* Bainsley — υποστηρίζει ελληνικά */
  font-style: italic; /* Bainsley italic = elegant visual contrast */
  color: var(--gold);
}
```
**Γιατί:** Το Bainsley italic σε gold δίνει οπτική διαφορά από το upright Bainsley body text, χωρίς να πέφτεις σε system fonts. Πολύ πιο ελεγμένο αποτέλεσμα.

## 7. ΕΠΙΠΛΕΟΝ ΠΑΡΑΤΗΡΗΣΕΙΣ

### 7.1 Scroll reveal — αργά animations

```css
/* ΤΡΕΧΟΝ */
.reveal { transition: all .8s cubic-bezier(.25,.46,.45,.94); }
```
**Παρατήρηση:** `.8s` είναι αρκετά αργό για elements που εμφανίζονται κοντά-κοντά (π.χ. trust cards). Δημιουργεί "waterfall lag" effect.

**Πρόταση:** Μείωσε σε `.65s` και χρησιμοποίησε `stagger` delays για sequential elements:
```css
.reveal { transition: all .65s cubic-bezier(.25,.46,.45,.94); }
.trust-card:nth-child(1).reveal { transition-delay: 0s; }
.trust-card:nth-child(2).reveal { transition-delay: .1s; }
.trust-card:nth-child(3).reveal { transition-delay: .2s; }
```

---

### 7.2 Hero description max-width

```css
/* ΤΡΕΧΟΝ */
.hero-description { max-width: 520px; }
```
**Παρατήρηση:** 520px είναι αρκετά στενό για μεγάλες οθόνες — δημιουργεί narrow column of text under a wide title.

**Πρόταση:** `max-width: 600px` για καλύτερη αναλογία.

---

### 7.3 Philosophy quote font size

```css
/* ΤΡΕΧΟΝ */
.philosophy-quote { font-size: clamp(1.35rem, 3vw, 2.35rem); }
```
**Παρατήρηση:** Στα `2.35rem` (desktop max) είναι εντυπωσιακό. Στα `1.35rem` (mobile min) χάνει λίγο impact.

**Πρόταση:** `clamp(1.45rem, 3vw, 2.5rem)` — ελαφρώς αυξημένο min για mobile.

```
Private Villas:  "Intimate dining in the comfort of your retreat, at the pace you set."  (11 words)

Exclusive Venues: "Blank canvas? We bring the concept, the kitchen, and the execution."  (11 words)

```
