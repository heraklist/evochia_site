# Evochia — Premium Event Catering & Private Chef Services

Static website for [evochia.gr](https://evochia.gr)

## Structure

```
├── index.html          Homepage
├── about.html          About Evochia
├── catering.html       Event Catering
├── private-chef.html   Private Chef
├── menus.html          Menu Collections
├── contact.html        Contact & Quote
├── css/site.css        Styles
├── js/site.js          Scripts
├── assets/
│   ├── logo.png
│   └── favicon.png
├── CREDITS.md
└── README.md
```

## Features

- Bilingual EN/EL with language toggle
- Preloader with logo animation
- Responsive design
- Floating conciergerie contact panel
- Scroll-reveal animations
- Photography in `photos/` folder (WebP format — see CREDITS.md for Unsplash attribution)

## Deployment

Deployed on Vercel with clean URLs (`vercel.json`). No build step.

## Brand

Palette: Dark green (#0a1f15), Gold (#c4a265), Cream (#f0e8d8)
Fonts: Cormorant Garamond + Raleway (Google Fonts)
Domains: evochia.gr, evochia.com (Papaki)
